Шаблон компонента bitrix:news.list

В рамках тестового задания был создан шаблон компонента bitrix:news.list.

Шаблон был разработан и протестирован в локальной установке Bitrix (XAMPP).  
В репозитории приведён только код шаблона компонента.

Что сделал
- создан пользовательский шаблон компонента news.list
- изменена разметка и базовые стили элементов новостей
- шаблон протестирован на локальном сайте

## Структура
Основные файлы шаблона:

- template.php — вывод списка новостей  
- style.css — стили шаблона  
- .parameters.php — параметры шаблона  
- lang/ — языковые файлы  

## Расположение в проекте Bitrix
Шаблон размещается в директории:
/local/templates/corp_services_blue/components/bitrix/news.list/my_component/
